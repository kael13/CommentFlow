🔒 Privacy Policy — Comment Flow

## 1. Data We Collect

Comment Flow may process the following data from Facebook:

- Public comments on connected Facebook pages
- User names and profile information (as provided by Facebook API)
- Message content within comments
- Engagement activity (likes, replies, timestamps)

We do not intentionally collect sensitive personal data beyond what is publicly available through Facebook interactions.

## 2. How We Use Data

We use collected data strictly to:

- Automate comment replies and moderation
- Identify leads and customer inquiries
- Improve analytics and engagement insights
- Power AI-based classification and responses

We do not sell user data or use it for unrelated advertising purposes.

## 3. Data Storage

Data may be stored temporarily in automation tools (e.g., n8n, databases, Google Sheets).
Businesses control their own storage destinations.
Users can configure or delete stored data via their system settings.

## 4. Data Sharing

We only share data when:

- Required for automation workflows chosen by the business (e.g., CRM, Google Sheets, Telegram)
- Required by law or legal compliance requests

We do not share data with third-party advertisers.

## 5. Security

We implement reasonable security measures to:

- Protect API access tokens
- Secure workflow automation systems
- Prevent unauthorized access to stored data

However, no system is 100% secure, especially when integrated with third-party platforms.

## 6. User Rights

Depending on applicable laws, users may:

- Request access to their data
- Request deletion of stored data
- Opt out of automated processing (via page settings)

## 7. Third-Party Services

Comment Flow may integrate with:

- Facebook Graph API
- n8n automation workflows
- Google Sheets / CRM tools
- Messaging services (e.g., Telegram)

Each service has its own privacy policy and data handling rules.

## 8. Changes to This Policy

We may update this Privacy Policy as the product evolves. Updates will be reflected within the platform or documentation.
